# HTML Vercel Template

Template pronto per deployare app HTML statiche su Vercel in 5 minuti.

## 📦 Cosa contiene

- `public/index.html` - File HTML di partenza (sostituiscilo con la tua app)
- `vercel.json` - Configurazione Vercel già pronta
- Struttura cartelle corretta per deploy immediato

## 🚀 Come usare questo template

### Metodo 1: Usa questo repository come template (consigliato)

1. Clicca sul pulsante verde **"Use this template"** in alto
2. Scegli **"Create a new repository"**
3. Dai un nome al nuovo progetto
4. Clicca **"Create repository"**
5. Sostituisci il contenuto di `public/index.html` con la tua app
6. Vai su Vercel e importa il repository
7. Deploy automatico ✅

### Metodo 2: Duplica manualmente

1. Crea nuovo repository su GitHub
2. Copia i file da questo template:
   - `vercel.json` nella root
   - `public/index.html` nella cartella public
3. Carica la tua app sostituendo `index.html`
4. Collega a Vercel e deploya
